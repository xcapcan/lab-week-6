What is an internet domain name?

An Internet domain name is a unique name of an organization or person on the Internet. The name is combined with a generic top-level domain (gTLD), such as .com or .org.

Why are domain names so important on the web?

Domain names make it easy for the users to fing the website they’re looking for or find the website with the answer they’re looking for. It also adds credibility to businesses if they have their own domain name.

Why do domain names have to be unique?

Because it makes you stand out among other websites and businesses, plus makes it easier for your customer to find you. It also has to be very similar or the same as the name of the organization, that way if anyone searches for it, it will be the top search.

What does it mean to “register” a domain name?

In order for a domain name to be visible on the Internet, it must first be registered with any one of hundreds of registrars, which will verify that the name is absolutely unique
