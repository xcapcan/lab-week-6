Project plan

Title

Little Fashion Closet

Goal

To inspire women to develop their personal fashion style and give them resourceful information about all the fashion trends of different kind of style.

Target Audience

Young women (ages 15-30) who are looking for tips on their clothing and fashion style.

Main Features

Web site with links that are categorized by style and seasons. There will be posts (that would theoretically be updated) as new trends come up and update. There will also be a signup for the email newsletter.

Personality

Inspiring, edgy, hipster, stylish.

The site will be a resource for people who are interested to learn more about fashion world.

Limiting Factors

I haven’t really thought of any major limitations, but screen readers should be fine if I put alts for the pictures, that way it will be available for everyone. Since it’s a younger audience, there shouldn’t be any limitation about the knowledge of using the device, but it should still be easy enough to understand.

Development guide

Site structure

  • Root Directory
    • images directory – contains jpgs and gifs
    • styles directory – contains all of the style sheets
    • index.html – home page for site, site purpose and overview
    • inspiration.html - gallery with different fashion looks
    • styling.html - source gallery with all kinds of different styles with links to other websites about the sites
    • trends.html - page with different seasons and trends with links to articles about them
    • newsletter.html - sign up for the newsletter

Media assets

General Guidelines

  • Images optimized as JPG using the Photoshop save for web feature to determine the best choice. Photos generally saved as JPG (winterlook.jpg, leatherjacket.jpg, etc.)
  • All photos will have links to the original source (if not clickable, then in alt)

Styles guide

Background

Background is a white tint

{background-color: #F2F2F2}

Default font

Text is black, 14 pixels

p { font-size: 14px; color: #black;}

Header

The header section is 235px tall, positioned along the top of each page with the logo centered in the frame. The background is bright yellow.

header {
height: 235px;
background-color: #F2C063;
background-image: url("/images/logo.png");
background-repeat: no-repeat;
background-position: center;
background-size: cover;
}

Navigation

The main navigation is 120px tall, positioned just below the header with a background color of Warm Red Tan and Dark Warm Red Tan color.

nav { height: 120px; background-color: #BF7C63; color: #8C594D; }

Images

Images will be over warm red tan background with no border

figure { background-color: 6699CC; }

img { padding: 1em; }

Links

Links will be most likely be in a form of a button. The buttons will be bright blue and on hover will change to yellow

button { color: #034AA6 }

button:hover { background-color: #F2C063 }
