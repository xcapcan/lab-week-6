Name: Little Closet - fashion trends

Audience: Young women (ages 15-30) who are looking for tips on their clothing and fashion style

Purpose: This is a source for women where it contains information about the new fashion trends. It acts kind of like a blog, but with specific categories for kinds of seasons and styles. It provides a collection of information and acts as a resource for those who are interested in fashion.

Similar webs :
  • https://www.thebudgetfashionista.com/
  • https://stylecaster.com/fashion/ (this one is more like an online magazine though)
  • https://www.whowhatwear.com/tags/blogger-style
Content :
  • Landing page with fashion photos
  • Nav about trends
  • Subsections with different seasons (FW2019, SS2020, etc.)
  • Form to sign up for a fake newsletter

Site personality: Inspiring, hipster, edgy, fashionable It should make the person feel inspired and passionate about improving their style

Limitations: I haven’t really thought of any major limitations, but screen readers should be fine if I put alts for the pictures, that way it will be available for everyone. Since it’s a younger audience, there shouldn’t be any limitation about the knowledge of using the device, but it should still be easy enough to understand.
